The Great RGB Color Guessing Game

Match the RGB code game

This project illustrates some basic DOM manipulation between HTML and Javascript. The object is to guess the color square based on the RGB code you are provided with. I created this as part of the tutorials I used to learn about web development but added the two harder modes myself.

Taxidermy Gallery

Taxidermy portfolio

I spent 15 years as a taxidermist in South Texas where I worked on many different species. The most common animal I worked on, almost on a daily basis, was your average White Tail Deer. Think of Bambi. That area in Texas is some of the best in the U. S. for trophy animals and I was able to work on truly unique and amazing animals. We also had plenty of exotic animals come through the shop which lead to me creating many customized works of art.

Into the Depths Podcast

Podcast about Camelot Unchained

I host a podcast live on Twitch where my friends and I discuss the kickstarted MMORPG Camelot Unchained. We talk about the design and what we are looking for it to be for us as players. We have gone on hiatus since the release of information about the game has slowed down but intend to resume the program once the game enters into a further development phase.
